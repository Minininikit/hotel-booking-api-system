# Installation & Setup
To get this project up and running locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone [https://github.com/Minininikit/hotel-booking-api-system.git](https://github.com/Minininikit/hotel-booking-api-system.git)
   ```
2. Navigate to the SDK folder to inspect the client code:
   ```bash
   cd hotel-booking-api-system/hotel-booking-sdk
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
