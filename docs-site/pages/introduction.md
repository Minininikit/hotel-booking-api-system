# Hotel Booking System API

Welcome to the official API documentation for the Hotel & Room management system.

This documentation provides all the necessary details to integrate with our server, perform full CRUD operations on hotels and rooms, and effectively utilize our pre-generated JavaScript SDK client.

### Quick Start
Use the sidebar menu to navigate through the project modules:

* **Authorization:** Understand how to secure your requests using Bearer JWT.
* **About:** View project metadata and access the SDK repository.
* **API Endpoints:** Read the interactive technical reference generated from our OpenAPI specification.
