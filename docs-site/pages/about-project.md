# About the Project & SDK
This system is engineered for managing hotel properties and room allocations (Hotel & Room entities).

### Project Resources:
* **Source Code & SDK Repository:** [GitHub Repository](https://github.com/Minininikit/hotel-booking-api-system)
* **API Specification:** Check out the `openapi.yaml` file in the root directory.

The SDK client was automatically generated for **JavaScript (ES6)** and features pre-built data structures (`Hotel`, `Room`) alongside service abstractions to streamline networking tasks.
